# 🌍 AI Language Translation Tool

An AI-powered Language Translation Tool built using **Python**, **Streamlit**, and **Deep Translator**. The application enables users to translate text between multiple languages through an intuitive and modern user interface.

Developed as part of the **CodeAlpha AI Internship Program**.

---

## 📌 Features

- 🌐 Multi-language Text Translation
- 🔄 Source and Target Language Selection
- 📜 Translation History
- 📥 Download Translated Text
- 🔢 Character Counter
- ⚠️ Input Validation and Error Handling
- 🎨 Modern and User-Friendly Interface
- 🏗️ Modular Project Architecture
- 🚀 Fast and Lightweight Web Application

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/CodeAlpha_Language_Translation_Tool.git
cd CodeAlpha_Language_Translation_Tool
```

### 2. Create Virtual Environment

```bash
python -m venv venv
```

### 3. Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Mac/Linux

```bash
source venv/bin/activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Run Application

```bash
streamlit run app.py
```

---

## 🚀 How It Works

1. Enter text in the input box.
2. Select the source language.
3. Select the target language.
4. Click the **Translate** button.
5. View the translated output.
6. Download the translated text if required.
7. Access previous translations through the history section.
